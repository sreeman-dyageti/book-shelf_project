## Book Shelf Project

Book Shelf is a full-stack web application that allows users to manage their personal book collection.

## 🚀 Features

- Add, Edit, Delete books (CRUD)
- PostgreSQL database persistence
- Author–Book relational database design
- Sort books by newest or rating
- ISBN validation using Open Library API (Axios)
- Book cover integration using Open Library Covers API
- Error handling for invalid ISBN and duplicate books
- Clean UI using EJS partials



## 🛠️ Tech Stack

- Node.js
- Express.js
- PostgreSQL
- EJS
- Axios
- Open Library API


## 📦 Installation
```bash
1. Clone the repository:

git clone https://github.com/sreeman-dyageti/book-shelf_project

2 .Install dependencies:

3. Make sure PostgreSQL is running.
